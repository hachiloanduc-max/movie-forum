# 🎬 影谈 CineTalk — 电影讨论社区

> 发现好电影 · 分享热评 · 连接影迷

一个 PC 端电影讨论网站，收录 12 部豆瓣高分经典电影，支持评分浏览、类型筛选、深度讨论和评论互动。

## ✨ 功能

- 🎞️ Hero 大屏电影轮播推荐
- 🖼️ 真实 TMDB 电影海报（全球 CDN）
- ⭐ 真实豆瓣评分数据
- 🔍 实时搜索（片名/导演/演员）
- 🏷️ 按类型筛选
- 💬 深度影评与讨论区
- 📝 评论互动系统
- 🌗 影院级暗色主题

## 🚀 部署

这是一个纯静态网站，可直接部署到 GitHub Pages：

1. Fork 或上传此仓库
2. Settings → Pages → Source: `main` → Save
3. 等待部署完成，访问 `https://你的用户名.github.io/仓库名`

也可以直接用浏览器打开 `index.html` 本地预览。
